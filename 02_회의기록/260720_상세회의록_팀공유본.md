**5. A-2 ADTV90 논의와 절충안**

거래정지일을 ADTV90 관측창에 포함하고 0으로 반영하면 평소 유동성과
거래가능 여부가 섞일 수 있다는 우려가 제기되었다. 이에 ADTV90을 주
인디케이터로 유지하되 거래정지 원인을 보조 필드와 기준일 현재 상태로
분리하는 안을 제시했다.

| **산출·기록**                                                                                                                                                         | **의미**                                                              | **분류**           |     |
|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------|--------------------|-----|
| **ADTV90**                                                                                                                                                            | 최근 90개 시장개장일 동안 실현된 평균 거래대금                        | 주 인디케이터      |     |
| **traded_days_90**                                                                                                                                                    | 실제 거래량이 발생한 날의 수                                          | 보조 기록          |     |
| **zero_volume_days_90**                                                                                                                                               | 정상 개장 중 거래량 0인 날의 수                                       | 보조 기록          |     |
| **halt_days_90**                                                                                                                                                      | 최근 90개 시장개장일 중 거래정지일 수                                 | 보조 진단          |     |
| **missing_days_90**                                                                                                                                                   | 원천자료 결측일 수                                                    | 품질 기록          |     |
| **current_trading_status**                                                                                                                                            | 기준일 현재 정상거래 또는 거래정지                                    | 별도 시장상태 규칙 |     |
