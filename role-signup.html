남은 작업 역할 지원 체크리스트

기준일 2026-07-21 · 담당 미정 작업만 · 이름을 적고 하고 싶은 작업을 체크한 뒤 저장하면 결과가 저장소(role-signup-data.json)에 일괄 기록됩니다. 이미 다른 사람이 지원한 항목에는 이름이 표시되고, 2명 이상 겹치면 중복으로 표기됩니다.
토큰 만드는 법 · 동작 방식 GitHub → Settings → Developer settings → Fine-grained personal access tokens → Generate new token → Repository access에서 AIQuant-3rd-project 선택 → Permissions에서 Contents: Read and write만 부여 → 생성된 토큰을 위에 붙여넣기. 토큰은 이 브라우저의 localStorage에만 저장됩니다.
저장을 누르면 내 이름의 선택 목록이 role-signup-data.json에 커밋됩니다(같은 이름으로 다시 저장하면 덮어쓰기 = 수정). 조회는 토큰 없이도 됩니다.
