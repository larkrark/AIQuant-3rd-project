## 2. 조정안

### 그대로 수용 제안 (충돌 없음)

- **C PIT 규칙**: 주담당 김근형 / 교차 권보성(Mandate·팀 결정 정합성 검토) — 김근형 1희망이며 지원 결과에 경합자 없음. 기존 건의표의 안건 C 표시는 이 구조로 대체
- **B 상태 데이터(B3~B6)**: 주담당 김근형 / 교차 추주원 — "후보 재선정이 아니라 시점·상태·식별이력 보완" 범위로 한정. B1·B2(거래소·보통주 정의)는 개인 작업이 아닌 회의 의결로 분리
- **백테스팅 실행**: 주담당 권보성 — 경합자 없음. 교차검토는 아래 QA 분리 원칙과 함께 결정

### 중복 해소 제안 (QA 5건 · 발표 2건: 권보성 vs 김민호)

작성자·검토자 분리 원칙을 적용하면 자연스럽게 나뉨:

| 항목 | 제안 | 근거 |
|---|---|---|
| 수기 검산·독립 재산출 (Q2·Q3) | **김민호 주** / 권보성은 대상 제공 | 백테스팅 작성자(권보성)가 자기 결과를 검산할 수 없음. 김근형 산출물(PIT·상태규칙)의 독립 재현도 김민호 — 본인도 타인 재현 요청 |
| 대표 검토일 지정·스냅샷 (Q1) | 검토일 지정은 회의 의결, 스냅샷 보존은 김민호 | 분포 QA(기존 건의)와 같은 기준일 사용 |
| 성과·위험, ETF 비교 (Q4) | 권보성 주 / 김민호 교차 | BM·환율 명세 연속성 |
| 승인 체크리스트 (Q5) | 권보성 진행 / 전원 확인 | 팀 승인 절차 성격 |
| 발표 (P1·P2) | 중복 해소 대신 **구간 분담**으로 전환: 김근형=공시근거·PIT·검토도구(본인 제안), 권보성=Mandate·BM·지수 산출, 김민호=QA·검증, 추주원=테마·기업조사 | 2인 중복을 4인 분담으로 확대 |

## 3. 작업량 확인 (배정 원칙 대비)

- 김근형: B+C 두 묶음 = "대형 묶음 2개 이하" 한계선. 본인 1희망이므로 존중하되, **회의록 작성 등 문서업무를 다른 구성원에게 이관**해야 성립
- 권보성: 팀장+Mandate·결정로그+백테스팅+Q4·Q5+H·J 준비까지 가면 초과 위험 — 섹션 4 일부는 다른 구성원 배정 권장
- 김민호: Q1~Q3+분포 QA+I 달력이면 "범위·완료조건 명확한 산출물" 조건에 부합하는 적정 수준
- 추주원: 현재 배정 없음 — 미제출 상태. B 교차·G 비교자료·발표 구간에 더해 인디케이터 구현 후보 1순위로 확인 필요
