**3. 산출물 ① adtv90_distribution — 유동성 분포**

**목적:** 한국·미국 시장별 ADTV90 하한을 정하기 위한 분포 자료. 시장별로
통화·규모가 다르므로 하나의 절대금액을 공통 적용하지 않는다 (7/15 확정).

**계산 규칙**

- **관측창:** 최근 90거래일 (달력일 아님). 기준일에서 거슬러 올라간
  실거래일 90개.

- **공식:** ADTV90 = 90거래일 일별 거래대금의 평균. 거래소 제공
  거래대금을 우선 사용하고, 없으면 비조정 종가 × 실제 거래량으로
  계산한다.

- **조정종가 사용 금지:** 거래대금은 실제 거래된 금액이므로 반드시
  비조정 종가를 쓴다. yfinance는 auto_adjust=False로 받아
  Close(비조정)와 Volume을 사용한다. 국내는 KRX 정보데이터시스템 또는
  pykrx의 거래대금 컬럼을 우선한다.

- **시즈닝 연동:** 유효 관측치가 90개 미만인 종목(예: SPCX)은 ADTV90을
  계산하지 않고 NA + 사유를 기록한다. 시즈닝 게이트와 유동성 게이트는
  별개 규칙이다.

- **통화 분리:** KRW와 USD를 환산하지 않고 각자 통화로 둔다. 하한도
  시장별로 따로 정한다.

**산출 형식 (xlsx 또는 csv, 시트 2개)**

**시트 1 — 종목별 값:**

|                                |                                          |
|--------------------------------|------------------------------------------|
| **열**                         | **내용**                                 |
| ticker / name / market / theme | 종목 식별 정보, 시장(KR/US), 소속 테마   |
| as_of_date                     | 계산 기준일                              |
| adtv90                         | 계산값 (KRW 또는 USD)                    |
| currency / unit                | 통화, 단위 (원 / 백만원 / USD 등 명시)   |
| valid_obs                      | 유효 관측치 수 (90 미만이면 adtv90은 NA) |
| source                         | 데이터 출처 (KRX / yfinance 등)          |

시트 2 — 시장별 분위수: 시장별(KR/US)로 후보풀 종목의 5 / 10 / 25 / 50 /
75 백분위수. 하한 후보 논의는 이 표에서 출발한다.

**완료 기준**

- 후보풀 32종목 전체가 한 행씩 존재 (NA 포함)

- 기준일·통화·단위·출처가 파일 안에 기록됨

- 분위수 표가 시장별로 분리됨

**4. 산출물 ② quick_ratio_distribution — 당좌비율 분포**

**목적:** 재무건전성 게이트의 하한(0.5? 1.0?)을 정하기 위한 분포 자료.
하한은 분포와 테마별 특성을 확인한 뒤 백테스트 전에 확정한다 (7/15
확정).

**계산 규칙**

- **정의 고정:** 당좌비율 = (유동자산 − 재고자산) ÷ 유동부채. 다른
  정의(현금성자산 기준 등)를 쓰면 반드시 명시한다. 정의가 섞이면 분포
  비교가 무의미해진다.

- **출처:** 국내 = DART 최근 분기·반기보고서 재무상태표. 미국 = EDGAR
  10-Q/10-K. yfinance의 quickRatio는 정의가 다를 수 있으므로
  참고용으로만 쓰고, 최소한 경계 종목은 원공시로 검증한다.

- **Point-in-Time 원칙:** 기준일 시점에 이미 공시된 최신 보고서만
  사용한다. 회계 기준일과 공시일을 모두 기록한다 — 백테스트에서 미래
  재무정보 참조를 막는 핵심 장치다.

- **업종 예외:** 금융회사·리츠는 적용 제외 명세를 확인한다 (현
  후보풀에는 해당 없음 예상 — 확인 후 '해당 없음'을 기록).

- **수주산업 메모:** 방산 등 수주산업은 운전자본 구조상 당좌비율이 낮게
  나올 수 있다 (7/15 논의). 우주/방산 테마의 분포가 다른 테마보다 낮으면
  그 사실을 메모로 남긴다 — 테마별 하한 차등 논의의 재료가 된다.

**산출 형식 (xlsx)**

|                                |                                               |
|--------------------------------|-----------------------------------------------|
| **열**                         | **내용**                                      |
| ticker / name / market / theme | 종목 식별 정보                                |
| fiscal_date / filing_date      | 회계 기준일 / 공시일 (PIT 검증용, 둘 다 필수) |
| quick_assets / current_liab    | 당좌자산, 유동부채 (통화·단위 명시)           |
| quick_ratio                    | 계산값                                        |
| source_url                     | DART/EDGAR 원문 링크                          |

\+ 요약: 전체 및 테마별 분위수(25/50/75)와 0.5 · 1.0 미만 종목 수.

**완료 기준**

- 32종목 전체에 회계 기준일·공시일·출처 링크가 채워짐

- 정의(재고자산 차감 방식)가 파일에 명시됨

- 테마별 분포 요약과 수주산업 메모 포함

**5. 산출물 ③ theme_exposure_table — 테마 노출도 근거표**

**목적:** 종목선정 규칙의 1번 게이트인 테마 노출도 판정의 근거 기록. "이
종목이 왜 이 테마인가"에 문서로 답하는 표다.

**구조:** 노출도 규칙은 **판정 로직**(테마 경계 정의 + 등급 기준)과
**방어 로직**(예외 처리 목록)으로 구성한다. 예외는 사전에 열거된 닫힌
목록만 인정하며, 모든 예외의 기본 처리는 보류 또는 하향(fail-closed)이다
— 예외가 특정 종목의 구제 통로가 되는 것을 차단하기 위함이다.

**선행 조건 0 — 테마 경계 정의 (7/22 회의 1번 안건)**

등급 판정 전에 각 테마가 어떤 사업·제품까지 포함하는지 열거형으로
정의한다. 경계 정의 없이 등급부터 매기면 판정자마다 다른 테마 지도를
들고 채점하게 된다. 최소한 아래 경계 질문 4개는 회의에서 닫는다:

- 우주·방산: 지상 방산(현대로템의 전차 등)을 포함하는가, 우주개발
  관련으로 한정하는가

- 에너지·전력: 발전 유틸리티(한국전력)와 수소연료전지(두산퓨얼셀)를
  기기·인프라와 같은 테마로 보는가

- AI·로보틱스: AI 플랫폼 서비스(NAVER)와 IT서비스(삼성SDS·LG CNS)를 AI
  시스템·로봇 제조와 같은 테마로 보는가

- 공통: 밸류체인 공급자(부품·소재·전선)는 어느 단계까지 포함하는가

**선행 조건 1 — 등급 정의 초안 (7/22 회의에서 확정)**

판정은 **2트랙**이다: 테마 매출비중이 공시로 확인되면 수치 기준(A트랙),
비율이 공개되지 않은 기업은 숫자를 임의로 만들지 않고 정성
기준(B트랙)으로 분류한다.

|          |                                                   |                                                         |
|----------|---------------------------------------------------|---------------------------------------------------------|
| **등급** | **A트랙 — 매출비중 공시 시**                      | **B트랙 — 비율 미공시 시 (정성)**                       |
| 직접     | 테마 관련 매출 비중 50% 이상 (세그먼트 매출 기준) | 테마 사업이 공시 문서상 주력 사업으로 명시됨            |
| 핵심지원 | 테마 매출 20~50%, 또는 밸류체인 핵심 공급자       | 테마 밸류체인 공급이 주요 사업부문 중 하나로 공시됨     |
| 제한     | 테마 매출 20% 미만이거나 부수 사업                | 테마 관련 사업이 언급되나 주력·주요 부문이 아님         |
| 근거부족 | —                                                 | 공시 문서로 테마 사업 자체를 확인할 수 없음 → 편입 불가 |

**B트랙 안전핀 2개:** ① 불확실하면 낮은 등급(보수 우선). ② 근거 신뢰도
'하'(IR·제품 자료뿐)만으로는 '직접'을 부여할 수 없다 — '직접'은 신뢰도
'중' 이상 근거가 필요하다.

**근거 자료 위계와 신뢰도 (7/15 확정 원칙 + 신뢰도 기록)**

사업보고서·10-K 세그먼트 매출 \> 수주공시 \> 제품·IR 자료. 기사 단독
인용으로는 편입할 수 없다. 근거는 원문 링크로 남기고, 근거 신뢰도를 별도
열에 기록한다:

|            |                                                               |
|------------|---------------------------------------------------------------|
| **신뢰도** | **근거 유형**                                                 |
| 상         | 사업보고서·10-K/10-Q의 세그먼트 매출 (감사·공시 대상 수치)    |
| 중         | 수주공시·계약공시 (공시 대상이나 매출비중 환산에 추정 개입)   |
| 하         | 제품 카탈로그·IR 자료·회사 홈페이지 (자체 홍보성, 검증 안 됨) |

**방어 로직 — 예외 처리 닫힌 목록 (E1~E6)**

아래 목록에 없는 예외는 인정하지 않는다 ("기타 예외는 판정자 재량" 조항
금지). 예외가 발동되면 근거표의 exception_applied 열에 번호와 사유를
기록한다. 어떤 예외도 등급을 올려주지 않는다 — 내리거나 보류시키기만
한다:

|        |                                                                       |                                                                              |
|--------|-----------------------------------------------------------------------|------------------------------------------------------------------------------|
| **\#** | **예외 상황**                                                         | **처리 규칙 (전부 fail-closed)**                                             |
| E1     | 테마 매출을 공시로 확인 불가                                          | 근거 위계 순차 적용(세그먼트→수주공시→IR), 전부 실패 시 근거부족 = 편입 불가 |
| E2     | 세그먼트 매출이 있으나 테마와 1:1 대응 안 됨 (효성중공업 건설 혼재형) | 보수적 추정(테마 매출 하한값) 사용 + 추정 근거 기록                          |
| E3     | 복수 테마 해당 (APH형)                                                | 매출 비중 큰 테마 1개 배정, 비중 확인 불가 시 판정 보류                      |
| E4     | 순수 지주회사 (한화·LS형)                                             | 연결 세그먼트 기준 판정, 테마 사업이 간접 보유뿐이면 '제한' 이하             |
| E5     | 판정 시점에 분할·합병·매각 공시 진행 중                               | 판정 보류, 완료 후 차기 정기변경에서 재판정                                  |
| E6     | 교차검토자와 등급 이견                                                | 낮은 등급 채택(보수 우선), 3인 합의 시에만 상향                              |

**산출 형식 (xlsx)**

|                              |                                                                            |
|------------------------------|----------------------------------------------------------------------------|
| **열**                       | **내용**                                                                   |
| ticker / name / market       | 종목 식별 정보                                                             |
| theme                        | 판정 테마 (우주·방산 / AI·로보틱스 / 에너지·전력)                          |
| grade                        | 직접 / 핵심지원 / 제한 / 근거부족                                          |
| track                        | A(매출비중 공시) / B(정성 판정)                                            |
| evidence / source_url        | 근거 요약 + 사업보고서·공시 원문 링크                                      |
| evidence_confidence          | 근거 신뢰도 상/중/하 ('직접' 등급은 '중' 이상 필수)                        |
| revenue_exposure             | 테마 관련 매출 비중 — 공시 기반일 때만 기입, 임의 추정 금지 (B트랙은 공란) |
| multi_theme                  | 복수 테마 해당 시 대표 테마와 판정 사유 (예: APH — 에너지 vs AI)           |
| exception_applied            | 발동된 예외 번호(E1~E6)와 사유, 없으면 공란                                |
| rater / rate_date / reviewer | 판정자 / 판정일 / 교차검토자                                               |

**작성 순서 — 경계 사례부터**

판정이 쉬운 종목이 아니라 어려운 종목부터 한다. 아래 11종목의 판정
논리가 서면 나머지는 자동으로 따라오고, 등급 기준의 결함도 여기서
드러난다:

- **미국 4:** 효성중공업(전력기기+건설 혼재 — 국내지만 동일 유형),
  APH(부품 공급자 + AI/에너지 중복), TER(테스트 장비 + 협동로봇 UR
  혼재), ATI(특수소재 공급자)

- **국내 7:** 한화(지주), LS(지주+전선), NAVER(AI 매출 비중?),
  삼성에스디에스(IT서비스 전반), 한국전력(발전 유틸리티 — 기기·인프라와
  성격 상이), 현대로템(지상 방산 — '우주개발' 포함 여부),
  두산퓨얼셀(수소 — 테마 정의 확장 여부)

**주의**

- 판정과 편입은 분리한다 — SPCX는 노출도 '직접'이라도 시즈닝 게이트로
  보류 중. 근거표에는 기록하되 편입 여부는 게이트가 정한다.

- Seed Basket 종목이라는 이유로 등급을 후하게 주지 않는다. 근거부족이면
  근거부족으로 적는다.

- 선행 확인: 국내 우주 3 · AI 3의 Seed Basket 선택 기록이 아직 로컬에
  없다. 근거표 착수 전에 팀 폴더에 기록한다.

**완료 기준**

- 후보풀 32종목 전체 판정 완료, 근거부족 포함

- 모든 행에 원문 링크 존재 (기사 단독 없음)

- 경계 사례 11종목에 판정 논리 서술 존재

- 중복 테마 규칙(E3)이 APH 사례로 실제 적용됨

- 모든 행에 트랙(A/B)·근거 신뢰도 기입, '직접' 등급 중 신뢰도 '하'인 행
  없음

- 예외 발동 행에 E번호·사유 기록, E1~E6 밖의 임의 예외 없음
