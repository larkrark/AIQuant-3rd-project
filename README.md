## 문서 수정 방법 (위키 방식)

1. GitHub 웹에서 md 파일 열기 → 연필 아이콘 → 수정 → Commit changes
2. 커밋하면 1~2분 내 위키에 자동 반영됩니다 (별도 빌드 없음)
3. docx 원본을 수정한 경우, 같은 이름의 .md 사본도 함께 갱신해 주세요 (md가 위키에 표시되는 판입니다)
4. 팀 결정이 걸린 내용은 반드시 [결정로그](01_운영문서/결정로그.md)에 기록합니다
