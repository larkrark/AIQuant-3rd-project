# analysis — 산출 러너 (비교시험·분포·재무·달력)

결정로그에서 **절차는 확정, 산출 코드는 예정**인 일회성 분석 러너 자리.

| 예정 항목 | 근거 |
|---|---|
| ADTV90 '0 반영 vs 분모 제외' 비교시험 | D-12 ⑤ |
| 시장별 ADTV90 분포(P5·P10·P25·P50·P75) 산출 | 룰북 8.1, #16 |
| 비교시험 8개 시나리오(K_A·K_B·c_w) | D-12 ⑦ |
| 당좌비율 확인표(QR·계산상태 3종) 산출 | D-12 ⑥ |
| 과거 2개 분기 달력 적용시험 | D-11 #30·D-12 ⑩ |

- engine의 `config`·모듈을 재사용해도 되지만(독립성 요구 없음), 성과 결과로 규칙을 바꾸지 않는다(R3).
- 산출물은 `../data/` 하위에 저장하고 `rule_version`을 함께 기록한다.
