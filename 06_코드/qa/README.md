# qa — 독립 재산출·검산 (김민호)

역할배정 확정본의 QA 방식 정의에 따른 **독립 재산출** 트랙.

> 독립 재산출 = **코드 비공유·규칙 문서(룰북·데이터사전·결정로그)만으로 별도 구현** 후 단계별 산출물을 engine 결과와 허용오차 내 비교.

## 독립성 규칙 (중요)

- 이 폴더의 코드는 **`engine/`을 import하지 않는다.** engine 코드를 재사용하면 "독립" 재산출이 아니다.
- 규칙 문서만 근거로 삼는다: `01_운영문서/{결정로그.md, 데이터사전.md, 260724_…룰북_v0.4…md}`.
- 비교 대상은 engine 산출물(`data/pilot_run/output*/`)이며, 불일치는 사유·규칙경로와 함께 기록한다.
- 수기 검산(J-5): 소표본(예: 1종목×5일 + 리밸 1회)을 사람이 직접 계산해 대조.

(비교시험·분포·재무 확인표 같은 "engine과 공유해도 되는" 산출 러너는 `analysis/`에 둔다.)
